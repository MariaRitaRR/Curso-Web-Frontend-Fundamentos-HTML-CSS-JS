## Sobre o curso

Este curso aborda as principais tecnologias para se tornar um(a) programador(a) web front-end: **HTML**, **CSS** e **JavaScript**.

Durante o curso, são aprofundados conceitos de:

- Semântica, SEO e acessibilidade em HTML
- Layouts modernos e responsivos com **Flexbox** e **CSS Grid**
- Efeitos visuais e animações com **CSS3**
- Interatividade, validação de formulários e manipulação do DOM com **JavaScript**

O foco é prático: os conteúdos são aplicados em projetos reais, incluindo a criação de sites responsivos do zero.

### O que este curso ensina

- Montar qualquer tipo de site estático (portfólio, site pessoal, apresentação de empresa)
- Criar sites **verdadeiramente responsivos**
- Construir layouts complexos com **Flexbox** e **CSS Grid**
- Aplicar efeitos visuais avançados com **CSS3**

### Pré-requisitos

Nenhum conhecimento prévio de programação — apenas noções básicas de sistema operacional.

### Para quem é

Qualquer pessoa que queira ingressar na área de **web design** e/ou **front-end**.
